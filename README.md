# 📸 Photo Gallery Website

This is a simple and responsive **Photo Gallery Website** developed using **PHP**, **HTML**, **CSS (Bootstrap 5)**, and **JavaScript**. It showcases a modular layout with clean design and customizable content.

---

## ✨ Features

- ✅ Responsive layout using Bootstrap 5
- ✅ Modular PHP pages (`index.php`, `about.php`)
- ✅ Custom CSS styling (`style.css`)
- ✅ Basic interactivity via JavaScript (`script.js`)
- ✅ Image gallery support (`images/` folder)

---

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript  
- **Backend**: PHP

---

## 🚀 Getting Started

To run this project on your local machine:

### 1. Clone the Repository
```bash
git clone https://github.com/jdilsham/Photo_Gallery_Website.git
```

### 2. Set Up a Local Server
Use a local server environment like XAMPP or WAMP:

- Move the project folder to your server directory:

  - htdocs/ for XAMPP

  - www/ for WAMP

### 3. Run the Website
- Start Apache from your local server control panel

- Open your browser and navigate to:
```bash
http://localhost/Photo_Gallery_Website/index.php
```
